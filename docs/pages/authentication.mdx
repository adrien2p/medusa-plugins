### What is included

This plugin provides you with the following authentication providers
- [Google](/authentication/google)
- [Facebook](/authentication/facebook)
- [Linkedin](/authentication/linkedin)
- [Auth0](/authentication/auth0)
- [Firebase](/authentication/firebase)
- [Azure AD](/authentication/azureoidc)

### Contribute

This plugin provides you with a set of external authentication providers. The purpose is to allow you to provide
other authentication systems than just the native email/password.

If you feel that you would like, **a specific provider to be added**, **report a bug** or just **contribute**

Then you can either:
- [create an issue](https://github.com/adrien2p/medusa-plugins/issues)
- [open a PR](https://github.com/adrien2p/medusa-plugins/pulls)
